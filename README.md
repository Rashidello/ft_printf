# ft_printf
This project has been created as part of the 42 curriculum by rarayano

## Description:
The objective of ft_printf is to recreate a printf function

## Instructions:
 to compile the project use command
 ```bash
 Make
 ```
 to run the program use command 
 ```bash
 main.c file_name.a
 ```
 ### useful commands:
 deletes compiled files:
 ```bash
 make clean
 ```
deletes all generated files:
```bash
make fclean
```
deletes all generated files then compiles everything again(hard reset)
```bash
make re
```
